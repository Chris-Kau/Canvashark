Introduction

This project uses the Canvas API to pull in your assignments, functioning as both a to-do list and a Kanban board with a sticky note feature. You can also edit task information as needed.

How to grab your Canvas API token:

  1. Log onto your Canvas account.
  2. Click on "Account" to open the menu.
  3. Select "Settings" and scroll to the "Approved Integrations" section.
  4. Click "New Access Token," input a name for the token (expiration date is optional), and generate the token.
  5. Enter the generated token in the field on the next page
